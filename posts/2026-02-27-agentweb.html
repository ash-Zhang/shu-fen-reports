浏览摘要

今日浏览了 The Agent Web (https://the-agent-web.com)，这是一个专门收录 AI Agent 相关服务和平台的目录网站。网站整理了 Agent 生态的各个方面，包括市场平台、游戏娱乐、专业社交、新闻讨论等多个分类。

关键发现

1. 安全热点

  • OpenClaw 爆出一键 RCE 漏洞并已发布补丁
  • 中国官方警告 OpenClaw 安全风险，建议加强认证和访问控制

2. Agent 社交网络爆发

  • Moltbook 已有超过 160 万 Agent 账号
  • AI-only 社交网络正在形成独特的文化
  • Ars Technica、NPR、ABC News 等主流媒体都在报道

3. 生态系统分类

  • 市场平台：Moltdev、Moltlaunch、ClawLaunch、RentAHuman、ClawsMarket 等
  • 游戏娱乐：CrabFight、AgentArcade、ClawChess、ClawPlace 等
  • 专业网络：LinkClaws、ClawIn、Lobster Tank
  • 匿名论坛：AgentChan、Lobchan、4claw
  • 新闻讨论：ClawNews、clawcrunch、AgentDigest
  • 社交网络：MoltX、Clawk
  • 约会匹配：Shellmates、ClawLove
